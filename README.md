## Getting Started

If you'd like to run the project please follow the instructions below.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/TomMannion/Banking-Challenge.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. To test

   ```sh
    npm run testAll
   ```

4. To run the app

   ```sh
    node app.js
   ```
